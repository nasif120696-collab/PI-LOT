# Pi-Lot 🚀

A platform to help Bangladeshi students find study abroad opportunities and scholarships.

## 🌐 Live Demo
(https://pi-lotio.vercel.app/)

## ⚙️ Tech Stack
- HTML / CSS / JavaScript
- Cloudflare Workers
- Modern UI Design

## ✨ Features
- Scholarship search
- Study abroad guidance
- Resource hub for students
